## 프로젝트 소개
몰려오는 먼지들을 박자에 맞춰 물 청소하는 리듬 액션 게임입니다. 

음악을 좋아하는 청소부 'Dustin'이 되어, 몰려오는 먼지 더미들을 씻어내야 합니다!

넥슨 ‘재밌넥’에서 2박 3일간 제작된 프로젝트로, 이후 폴리싱 작업을 거쳐 무료 게임으로 출시하였습니다.
